margin:auto实现绝对定位元素的水平垂直居中

CSS3实现方法就是使用transform 

.element {
    position: absolute;
    left: 50%; 
    top: 50%;
    transform: translate(-50%, -50%);    /* 50%为自身尺寸的一半 */
}

使用margin:auto方法

.element {
    position: absolute; 
    left: 0; 
    top: 0;
    right: 0;
    bottom: 0;
    margin: auto;    /* 有了这个就自动居中了 */
}

该方法IE8+以及其他浏览器都是OK的.

DEMO

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
    .demo{
        width:200px;
        height:40px;
        background:red;
        position: absolute;
        left: 0;
        top: 0;
        right: 0;
        bottom: 0;
        margin: auto;
    }
    </style>
</head>
<body>
    <div class="demo">test</div>
</body>
</html>
